# 插入数据时，判断表中是否存在数据

```sql
INSERT INTO `sys_dict` ( `name`, `value`, `type`, `description`, `sort`, `remarks`, `company_id`, `a_id` ) SELECT
'无房客户',
'user_type_usernull',
'user_type',
'住户类型',
6,
NULL,
NULL,
NULL 
FROM
DUAL 
WHERE
	! EXISTS (
	SELECT
		1 
	FROM
		sys_dict 
	WHERE
		`value` = 'user_type_usernull' 
	AND type = 'user_type' 
	)
```
